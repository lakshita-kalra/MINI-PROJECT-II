Smart Digital Wardrobe

A Full-Stack Web Application that suggests outfits based on real-time weather and your personal wardrobe. This project provides an interactive, user-friendly, and automated way to manage wardrobe items and get outfit recommendations.

🔗 Live Demo: https://digitalsmart.netlify.app/

About the Project

Smart Digital Wardrobe combines weather data with your wardrobe to provide daily outfit suggestions. Users can manage their clothes, 
get outfit recommendations, and enjoy an interactive UI experience.

The system offers features for:

Users – Add, remove, and view wardrobe items; get outfit suggestions based on weather.  
Weather Integration – Receive real-time outfit recommendations based on current weather conditions.  

Features

✅ User Authentication – Secure login and signup with profile management.  
✅ Wardrobe Management – Add, remove, or view wardrobe items.  
✅ Real-Time Weather Suggestions – Fetches current weather and suggests appropriate outfits.  
✅ Interactive UI & Animations – Smooth transitions and engaging user experience.  
✅ Backend Integration – Stores user and wardrobe data using Supabase.  
✅ Responsive Design – Optimized for mobile and desktop screens.  

 Tech Stack

**Frontend:** React, TypeScript, Tailwind CSS, Framer Motion  
**Backend:** Supabase  
**APIs:** Weather API, Wardrobe API  
**Deployment:** Netlify  

Usage

Open the live app here: [Smart Digital Wardrobe](https://digitalsmart.netlify.app/)  
- Login or signup as a user.  
- Add your clothes to your wardrobe.  
- View outfit suggestions based on real-time weather.  
- Enjoy smooth animations and responsive UI.  

🔮 Future Scope

- Integrate AI-based outfit recommendations.  
- Add multi-user support with shared wardrobe suggestions.  
- Enable advanced filters for wardrobe items.  
- Push notifications for outfit suggestions on mobile devices.
